This is my downloads section:(I included Liero as well as the programs!)

Liero Version 1.33
Here's what you're here for. Named the greatest freeware strategy computer game by 4 out of 5 Worms players. Why don't you join the thousands of Liero players and get yourself a copy of the latest version, 1.33. Liero runs in DOS format and many computers have problems running it. Liero is shareware, meaning it is completely free.
LieroKit 1.51
LieroKit lets you alter the weapons, objects, the AI (but not much on the AI), and lets you create sound banks, weapons, and more! It doesn't let you make Liero Levels like in wormhole, though. And if you're getting this to make sounds, you should probably go with SND Tools.
Snd Tools v1.3
This is one of the first utils towards extreme liero modding, made by Tim Verweij. This tool lets you modify the liero soundbank, adding and extracting sounds is made easy with this excellent util. The latest version fixes some bugs. (136kb)
LieroHacker 0.8u
This program by Erik Lindros can change a lot of different aspects of the game. This program hacks the liero.exe to create cool effects in the game. New hacks can be added easily. (33kb)
This is the all new version of the first custom made Liero level maker. It has plenty of new features and its own editor. I suggest you download and try it immediately cause it simply rocks. All this thanks to Johan Otterud. Have fun and if you make any kickass levels don't forget to email them to us so we can post them here. (1.83Mb)
Liero Blood Increaser
This is a nice tool that lets you increase the amount of blood in the game up to 32750% (10kb)
MoreBlood
MoreBlood's purpose is for those who like blood in Liero. Personally, I think the 500 percent limit standard with Liero is enough, but some people actually like stuff this sick.
Liero Graphics Editor
This utility let's you repaint the different objects in the game such as weapon projectiles, in-game explosions and of course the worms! Check out the Mods section to download Liero Graphic Mods (Skins). If you get an error trying to run this utility try downloading and then installing Wormhole v1.5 and it should work. (40kb)
LieroM8 B3
This is a graphical Liero Level Viewer / Activator made by Zander Zoft. It has some nice features which include adding levels to zip's, increasing blood and changing/saving palletes. It activates mods (no LWP) and levels. It converts some stuff too. The newest version includes more interesting hack settings. (negative gravity for example) (283kb)
Liero Stuff Activator v0.87c
This program is the successor of the "Powerlevel activator for DOS". Currently it activates levels, powerlevels, graphic mods (CHR), soundbanks (SND), fonts (LFT), palettes (LFT/PAL) and last but not least, LieroKit Weapon plugins (LWP)! The program can also extract the palette, weapons (to LWPs) and the font. There is no real weapon editor yet, but you can raw edit the LWP files using a normal text editor. (56kb)
Liero Editor version 2.50.3C
This DOS editor features the best CHR editor yet! It also contains a very good level editor. Use left and right arrow keys in the main menu to select English. Includes a readme in English now. :) Made by fra.iesus. (59kb)
Lev Tools v1.1
With Liero LevTools you can convert windows bitmaps (504x350 24bit) to Liero Levels and back again. (Quoted from the readme of levtools) Its basically a more simple Level Editor which shows more than 190 colors available within Liero to build levels with. Included is a helping BMP. Also very good to make level previews. Enjoy and thanks to Tim Verweij. (Probably about 90% compatible with Wormhole 1.5 generated Levels) (298kb)
Liero Level Preview Maker
This is the program I use to make the level previews on the 'Levels' download page. There is no readme/manual of any kind, but it's simple to use. This is the only program that converts levels and powerlevels to bitmaps AND adds shadows like they would look in Liero. Shadows can be turned on or off. Converts all levels to bmp when started normally. Can convert one level if its icon is dropped on a shortcut to this program. The only downside is that the program is slow. I uploaded this for all other Liero webmasters. (42kb)
CharEd
This program lets you edit the text of the Liero menu, suicide message, kill message and more! One great program if you intend to make TCs. Put this program in your Liero directory to locate Liero more easily.
